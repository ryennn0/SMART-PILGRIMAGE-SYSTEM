# Smart Pilgrimage System

Platform digital terintegrasi untuk pengelolaan kawasan wisata religi Kota Pasuruan:

- Payung Madinah
- Makam K.H. Mbah Hamid
- Alun-Alun Kota Pasuruan

## Fitur
- Smart Information
- Smart Crowd Management
- Smart UMKM Integration
- Smart Feedback & Governance

## Teknologi
- HTML5
- CSS3
- JavaScript

## Cara Deploy ke GitHub Pages
1. Upload project ke repository GitHub
2. Masuk ke Settings
3. Pilih Pages
4. Pilih branch `main`
5. Save

Website akan aktif dalam beberapa menit.
